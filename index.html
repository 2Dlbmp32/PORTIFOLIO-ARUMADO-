MEUS PROJETOS

“aprenda Xadrez”

O Clube de Xadrez Escolar é um site criado com o objetivo de facilitar o aprendizado do xadrez de forma prática, divertida e acessível. Aqui, você encontra explicações claras sobre regras básicas, conceitos essenciais, estratégias e aberturas famosas, ideais tanto para quem está começando quanto para quem já joga e deseja evoluir. A proposta é ensinar e motivar novos jogadores, incentivando o raciocínio lógico e matemático por meio de uma página web leve, organizada e acessível para todos os dispositivos. Mais do que um site, o projeto representa o esforço de alunos apaixonados pelo jogo que querem compartilhar conhecimento com toda a escola.

chessflix: historia de Magnus Carlsen

O ChessFlix é um site desenvolvido com o objetivo de tornar o xadrez mais acessível, interessante e envolvente para todos os públicos. A plataforma reúne conteúdos sobre a trajetória de Magnus Carlsen, o maior enxadrista da atualidade, e busca inspirar novos jogadores por meio de uma apresentação clara, moderna e educativa.Mais do que apenas uma página sobre xadrez, o ChessFlix é um projeto feito por alunos apaixonados pelo jogo, combinando tecnologia e conhecimento para aproximar a comunidade escolar desse universo estratégico. Com um design simples e responsivo, o site foi construído em HTML e CSS, garantindo compatibilidade com todos os dispositivos e facilidade de navegação. A proposta é mostrar como o xadrez pode ser empolgante, valorizando não só as jogadas e campeonatos, mas também as histórias por trás dos grandes mestres. O ChessFlix nasce como um convite para pensar, aprender e se encantar com o poder do raciocínio lógico através do xadrez.

“Rumo ao Vestibular”

O site “Rumo ao Vestibular” foi criado com o objetivo de ajudar estudantes do 3º ano do Ensino Médio a revisarem os conteúdos que mais caem nos principais vestibulares do país. Aqui, você encontra explicações organizadas por matéria, cobrindo os temas essenciais de Matemática, Química, História, Filosofia e Português — tudo de forma clara, prática e acessível. A ideia do site é tornar a preparação para o vestibular mais leve e eficiente. Cada matéria possui uma seleção dos tópicos mais cobrados, acompanhada de links confiáveis para estudo, resumos diretos e imagens ilustrativas, o que facilita tanto a compreensão quanto o foco nos estudos. O conteúdo foi pensado especialmente para alunos que desejam reforçar seu aprendizado com autonomia e organização. Seja para revisar antes das provas, reforçar algum conteúdo específico ou montar um cronograma de estudos, o “Rumo ao Vestibular” é uma ferramenta prática feita por estudantes para estudantes — com o apoio da tecnologia e do compromisso com a educação.

O projeto Clube de Xadrez Escolar é uma página web desenvolvida com HTML e CSS, com o objetivo de ensinar os fundamentos do xadrez de forma acessível, clara e envolvente. A ideia central é oferecer um ambiente online onde iniciantes e estudantes interessados possam aprender desde as regras básicas até as aberturas mais populares do jogo

A estrutura do site foi criada utilizando HTML semântico, que organiza as informações de maneira lógica e compreensível. Já o CSS é responsável por toda a parte visual, definindo as cores, a tipografia e o layout da página para proporcionar uma navegação agradável e intuitiva em diferentes dispositivos.

O site apresenta uma introdução sobre o clube e os objetivos do projeto, seguida por seções didáticas que explicam conceitos fundamentais, estratégias básicas, aberturas clássicas como a Ruy Lopez e a Defesa Siciliana, além de uma área dedicada às regras oficiais do xadrez. Todo o conteúdo é apresentado com linguagem direta, com foco no desenvolvimento do raciocínio lógico, matemático e estratégico dos alunos.

O estilo visual do site é moderno e limpo, com o uso de cores suaves, tipografia amigável importada do Google Fonts e variáveis CSS para manter a consistência no design. O layout é totalmente responsivo, garantindo uma boa experiência tanto em computadores quanto em celulares.

O projeto chessflix é uma página web desenvolvida com HTML e CSS, com o objetivo de apresentar de forma acessível e envolvente a trajetória de Magnus Carlsen, considerado por muitos o maior enxadrista da história. A proposta é oferecer uma experiência visual simples e informativa, destacando os principais marcos de sua carreira e seu impacto no mundo do xadrez.

Toda a estrutura do site foi construída utilizando HTML, que organiza o conteúdo de maneira lógica e bem distribuída. O CSS é responsável pela identidade visual da página, aplicando cores equilibradas, fontes modernas e uma disposição agradável dos elementos. O design responsivo garante que o conteúdo possa ser acessado com qualidade em diferentes dispositivos, como celulares, tablets e computadores.

A página destaca momentos importantes da carreira de Carlsen, como sua ascensão precoce, o título de Grande Mestre ainda na adolescência, as conquistas no Campeonato Mundial e seu estilo de jogo conhecido por combinar precisão, criatividade e resistência. Imagens ilustrativas e uma linguagem objetiva tornam o conteúdo mais atrativo, especialmente para estudantes e novos admiradores do esporte.

A estética do site foi pensada para refletir a elegância do próprio jogo de xadrez, utilizando tipografia leve importada do Google Fonts e organização visual limpa. O projeto demonstra como é possível unir conteúdo histórico e inspiração esportiva em um site funcional feito apenas com HTML e CSS.

“Rumo ao Vestibular”

“Rumo ao Vestibular” é uma página web criada com HTML e CSS com o objetivo de organizar de forma clara, acessível e atrativa os principais conteúdos do 3º ano do Ensino Médio que mais caem nos vestibulares e no ENEM. O projeto visa auxiliar estudantes na revisão das matérias essenciais, oferecendo uma experiência leve, informativa e visualmente agradável.

Construído em HTML, o site apresenta seções dedicadas a cinco disciplinas principais: Matemática, Química, História, Filosofia e Português. Em cada seção, o visitante encontra uma lista com os temas mais cobrados nos exames, uma imagem representativa e links diretos para sites confiáveis onde pode aprofundar o estudo, como Stoodi, Brasil Escola, Toda Matéria e Prepara ENEM.

O CSS foi utilizado para estilizar cada bloco de conteúdo com cores suaves, tipografia moderna e ícones temáticos, criando uma identidade visual amigável e responsiva. O layout se adapta bem a celulares, tablets e computadores, garantindo que o estudante tenha uma boa navegação em qualquer dispositivo.
